1. Description
--------------
Beyond Compare is a file and folder comparison utility for Windows.  Use it to
visualize and reconcile differences, merge changes, and synchronize folders.


2. System Requirements
----------------------
This program runs under:
- Windows Server 2025
- Windows 11
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows 10, Windows 10 x64

Mac and Linux versions are also available on our website.
